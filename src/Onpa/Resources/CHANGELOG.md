# Changelog

All notable changes to Onpa are documented here. This file is generated automatically by [semantic-release](https://github.com/semantic-release/semantic-release) on each release.

## Development Build

You're running a local development build of Onpa. Release notes are only embedded in builds produced by the GitHub release pipeline.

To preview a real changelog locally, run `npm run changelog:local` from the repository root, then rebuild the iOS app.
